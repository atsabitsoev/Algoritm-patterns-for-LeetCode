10. Binary Tree Traversal

Binary Tree Traversal — паттерн для обхода бинарного дерева различными способами: in-order, pre-order, post-order, level-order. Используется для поиска, сортировки, анализа структуры дерева.

Когда использовать?

Когда задача связана с деревьями: поиск, подсчёт, преобразование, сериализация, восстановление структуры и др.

Объяснение

  • In-order (лево-корень-право): используется для получения отсортированной последовательности из BST.
  • Pre-order (корень-лево-право): полезен для копирования дерева, сериализации.
  • Post-order (лево-право-корень): применяется для удаления дерева, вычисления высоты.
  • Level-order (BFS): обход по уровням, часто реализуется через очередь.
  • Обходы могут быть реализованы рекурсивно или с помощью стека/очереди.
  • Паттерн встречается во многих задачах на деревья: поиск пути, балансировка, подсчёт листьев и др.

Пример задачи

Вход: root = [1,null,2,3]
Задача: In-order обход дерева.
Выход: [1,3,2]

Swift пример

 class TreeNode {
    var val: Int
    var left: TreeNode?
    var right: TreeNode?
    init(_ val: Int) { self.val = val; self.left = nil; self.right = nil }
}

func inorderTraversal(_ root: TreeNode?) -> [Int] {
    var result = [Int]()
    func dfs(_ node: TreeNode?) {
        guard let node = node else { return }
        dfs(node.left)
        result.append(node.val)
        dfs(node.right)
    }
    dfs(root)
    return result
}

Дополнительный пример

Вход: root = [3,9,20,null,null,15,7]
Задача: Level-order обход дерева.
Выход: [[3],[9,20],[15,7]]

Советы и типичные ошибки

  • Не забывайте проверять на nil перед рекурсией.
  • Для level-order используйте очередь (Queue).
  • Внимательно относитесь к порядку обхода (in/pre/post).
  • В задачах на сериализацию/десериализацию учитывайте null-узлы.
