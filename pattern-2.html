2. Two Pointers

Two Pointers — техника, при которой два указателя проходят по массиву для поиска пар или элементов, удовлетворяющих условию.

Когда использовать?

Когда нужно найти пары в отсортированном массиве или выполнить задачу с двумя концами массива.

Пример задачи

Вход: nums = [1, 2, 3, 4, 6], target = 6
Выход: [1, 3]

Объяснение

  • Два указателя (обычно left и right) позволяют эффективно искать пары или диапазоны в массиве, особенно если он отсортирован.
  • Вместо вложенных циклов (O(n^2)) можно пройти массив за один проход (O(n)), двигая указатели навстречу друг другу.
  • Если сумма элементов меньше target — сдвигаем левый указатель вправо, чтобы увеличить сумму.
  • Если сумма больше target — сдвигаем правый указатель влево, чтобы уменьшить сумму.
  • Если сумма равна target — нашли нужную пару.
  • Техника применяется не только для поиска суммы, но и для поиска палиндромов, сортировки, удаления дубликатов и других задач.
  • Пример: задача "3Sum" использует два указателя внутри внешнего цикла для поиска трёх чисел с заданной суммой.
  • Два указателя могут двигаться как навстречу друг другу, так и в одном направлении (например, для поиска подстрок без повторяющихся символов).

Swift пример

 func twoSum(_ nums: [Int], _ target: Int) -> [Int]? {
    var left = 0
    var right = nums.count - 1
    while left < right {
        let sum = nums[left] + nums[right]
        if sum == target {
            return [left, right]
        } else if sum < target {
            left += 1
        } else {
            right -= 1
        }
    }
    return nil
}

let nums = [1, 2, 3, 4, 6]
let result = twoSum(nums, 6) // [1, 3]
