8. Overlapping Intervals

Overlapping Intervals — паттерн для работы с пересекающимися интервалами. Часто используется для объединения, поиска или удаления перекрывающихся диапазонов.

Когда использовать?

Когда задача связана с временными слотами, диапазонами чисел, отрезками на прямой, расписаниями и т.д.

Объяснение

  • Сначала интервалы сортируются по началу (или концу).
  • Затем перебираем интервалы и сравниваем текущий с предыдущим.
  • Если интервалы пересекаются, объединяем их в один.
  • Если не пересекаются — добавляем текущий в результат.
  • Паттерн часто используется для задач на расписания, обработку событий, анализ диапазонов.
  • В Swift удобно использовать массивы и сортировку, а также структуры для представления интервалов.

Пример задачи

Вход: intervals = [[1,3],[2,6],[8,10],[15,18]]
Выход: [[1,6],[8,10],[15,18]]

Swift пример

func merge(_ intervals: [[Int]]) -> [[Int]] {
    guard !intervals.isEmpty else { return [] }
    let sorted = intervals.sorted { $0[0] < $1[0] }
    var result = [sorted[0]]
    for interval in sorted.dropFirst() {
        if interval[0] <= result.last![1] {
            result[result.count-1][1] = max(result.last![1], interval[1])
        } else {
            result.append(interval)
        }
    }
    return result
}

Советы и типичные ошибки

  • Не забывайте сортировать интервалы перед обработкой.
  • Проверьте, что интервалы не пусты.
  • Внимательно относитесь к граничным случаям (например, интервалы с одинаковым концом и началом).
  • В задачах на удаление перекрытий используйте похожий подход, но с учётом удаления.
