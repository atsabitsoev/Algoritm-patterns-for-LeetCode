1. Prefix Sum

Prefix Sum — это техника предварительной обработки массива для быстрого вычисления суммы подмассивов.

Когда использовать?

Когда нужно многократно находить сумму элементов на отрезке массива или вычислять кумулятивные суммы.

Пример задачи

Вход: nums = [1, 2, 3, 4, 5, 6], i = 1, j = 3
Выход: 9

Объяснение

  • Префиксные суммы позволяют быстро находить сумму элементов на любом отрезке массива за O(1) после предварительной обработки.
  • Сначала строится массив P, где P[i] — сумма всех элементов от начала до i.
  • Чтобы узнать сумму на отрезке [i, j], достаточно вычислить P[j] - P[i-1] (если i > 0), иначе P[j].
  • Это особенно полезно, если нужно отвечать на множество запросов на сумму подотрезков.
  • Префиксные суммы применяются не только для суммы, но и для других операций (например, подсчёта количества элементов, подсчёта чётных/нечётных и т.д.).
  • Пример: если нужно узнать сумму чисел с 2 по 4 (включительно) в массиве [1,2,3,4,5,6], то это P[4] - P[1] = 15 - 3 = 12.
  • Префиксные суммы часто используются в задачах на динамическое программирование, анализ подстрок, работу с матрицами (2D prefix sum) и др.

Swift пример

 func prefixSum(_ nums: [Int]) -> [Int] {
    var result = [Int](repeating: 0, count: nums.count)
    result[0] = nums[0]
    for i in 1..<nums.count {
        result[i] = result[i-1] + nums[i]
    }
    return result
}

let nums = [1, 2, 3, 4, 5, 6]
let prefix = prefixSum(nums)
// Сумма от 1 до 3: prefix[3] - prefix[0] = 9

Советы и типичные ошибки

  • Не забывайте, что индексация в Swift начинается с 0.
  • Проверьте, что i > 0 перед вычислением P[j] - P[i-1].
  • Префиксные суммы можно использовать для поиска количества элементов, а не только суммы.
  • Внимательно относитесь к границам массива, чтобы избежать выхода за пределы.
